DB_STRING=mongodb://host.docker.internal:27017/local_library
DB_STRING=db-1://host.docker.internal:27017/local_library

On WSL (Windows Subsystem for Linux), use:
DB_STRING=mongodb://host.docker.internal:27017/local_library
This is the correct hostname for WSL to reach the host machine's Docker daemon.

However, if your MongoDB container is on the same Docker network (not the host), use the container name instead:
DB_STRING=mongodb://mongodb:27017/local_library
